# gh-agents\n\nGitHub App agents and integrations.\n\n## Purpose\n\nThis repository contains OpenClaw agent configurations and custom skills for interacting with GitHub Apps, automating workflows, and managing repository operations programmatically.\n\n## Structure\n\n- `agents/` — agent templates and configurations\n- `skills/` — reusable skills for GitHub interactions\n- `docs/` — documentation and setup guides\n\n## Getting Started\n\n1. Ensure your GitHub App is installed on the target organization.\n2. Configure credentials in `openclaw.json` or use the `github_jwt.sh` helper.\n3. Explore the agent templates to customize behavior.\n\n---\n\n_Last updated: 2026-03-26 by Spark_
