# gh-agents

GitHub App agents and integrations.

## Purpose

This repository contains OpenClaw agent configurations and custom skills for interacting with GitHub Apps, automating workflows, and managing repository operations programmatically.

## Structure

- `agents/` — agent templates and configurations
- `skills/` — reusable skills for GitHub interactions
- `docs/` — documentation and setup guides

## Getting Started

1. Ensure your GitHub App is installed on the target organization.
2. Configure credentials in `openclaw.json` or use the `github_jwt.sh` helper.
3. Explore the agent templates to customize behavior.
